# Desktop Renderer

This directory contains the real Electron renderer layer for the tray shell.

Current scope:

- `dock.html` and `dock.js` for the always-on-top drag-and-drop launcher
- `console.html` and `console.js` for the main workspace shell
- `overlay.html` and `overlay.js` for the compact quick-entry surface
- `preload.cjs` for the safe renderer bridge
