# Root Markdown Public Review

This file is the release-facing decision log for tracked Markdown files that
live at the repository root and are not standard community files such as
`README.md`, `CONTRIBUTING.md`, `SECURITY.md`, or `THIRD_PARTY_LICENSES.md`.

Current public-release policy: README is the only product-facing root Markdown
entry. Task plans, audit logs, cleanup notes, and product planning discussions
must live outside the tracked public root, typically under local ignored
`internal/` while they are still useful.

`verify:github-readiness` now fails if any non-standard root Markdown file is
tracked. This table should therefore remain empty.
