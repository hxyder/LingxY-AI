# 审计会议纪要 — 信息提取报告

## Owner（负责人）
**Desktop Runtime Team**（桌面运行时团队）

## Goal（目标）
**降低首次可见输出延迟，同时保持严格的工具策略。**
> Reduce first visible output latency while keeping tool policy strict.

## Follow-up（后续行动）
**在真实的 API 报告中添加 token 和 timing waterfall 字段。**
> Add token and timing waterfall fields to real API reports.
