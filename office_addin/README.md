# Office Add-in

Current scaffold includes:

- Word / Excel / PowerPoint manifests
- shared Task Pane HTML and bridge scripts
- transport strategy banner for `https://localhost:9413` vs `uca://office-submit`
- desktop Console one-click setup that registers the local Shared Folder catalog

Phase 4 base currently selects the protocol-handler fallback as the default ship
path after the add-in is loaded from the local catalog.
